About

This is a quickly whipped together, static webpage that tries to load flashcards and their metadata (via service workers web APIs) for offline use. I've written it with my phone in mind, and flashcards are created by uploading images to my CDN at /cards. If successful, you should be able to study flashcards without an internet connection, once you've loaded them at least once with an internet connection.

Source Code

You can read the sourcecode for this page, and some of the scripts to generate the flashcards, at https://github.com/jzacsh/study or clone directly via: git clone https://github.com/jzacsh/study.git
